== Description ==

**WPRobo Engage** helps you build and launch conversion campaigns — popups, floating bars, and slide-ins — without writing a single line of code.

= Features =

* **Unlimited Campaigns** – Create as many campaigns as you need
* **Popup Campaigns** – Exit-intent, timed, and scroll-based popups
* **Floating Bar** – Sticky header/footer notification bars
* **Slide-In Campaigns** – Corner slide-in opt-in forms
* **Template Library** – 10+ pre-built templates to get started fast
* **Smart Display Rules** – Show or hide campaigns by page, post, category, tag, or URL
* **Basic Analytics** – Track impressions and conversion rates
* **Lead Management** – View and export collected leads
* **Third-Party Form Embed** – Paste embed code from Mailchimp, ConvertKit, AWeber, and more
* **Import / Export** – Transfer campaigns between sites as JSON
* **Role-Based Access** – Control which admin roles can manage campaigns

= Source Code =

The unminified source files for this plugin are available on GitHub:

Build instructions: `npm install && npm run build`

== Installation ==

1. In your WordPress admin, go to **Plugins > Add New**
2. Search for "WPRobo Engage"
3. Click **Install Now** and then **Activate**
4. Navigate to **WPRobo Engage** in your admin menu to create your first campaign

Or manually:

1. Download the plugin ZIP file
2. Go to **Plugins > Add New > Upload Plugin**
3. Upload the ZIP and activate
